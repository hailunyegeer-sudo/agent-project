# 项目开发阶段的 Agent 使用说明

## 背景

本项目在开发阶段引入了 Agent 协助处理文件阅读、信息检索、格式转换和结果汇总。这里的 Agent 主要负责根据用户输入选择合适工具，并将工具返回内容组织为最终回复。

## 开发工作安排

开发工作包括环境配置、目录管理、日志记录、JSON 输入输出设计和异常处理。为了保证每个模块都能独立演示，代码中的 B1、B2、B3、B4、B5 分别保留命令行入口。测试时可以单独运行 Skill，也可以通过完整运行时观察消息序列和工具结果。

## 关键词关系

本文会分别出现 Agent 和 开发 两个词，但它们不构成连续的“Agent开发”短语。换言之，本文讨论的是“项目开发阶段如何使用 Agent”，而不是专门介绍 Agent开发 方法论。
